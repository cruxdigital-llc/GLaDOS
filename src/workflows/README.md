# Workflows

Workflows are the entry points for Agent interaction. They define the high-level steps for specific tasks.

## Categories

### Greenfield / Setup
-   **`mission.md`**: define the product mission.
-   **`plan_product.md`**: Establish the roadmap and tech stack.
-   **`establish_standards.md`**: Extract tribal knowledge into written standards.

### Feature Lifecycle (Split)
-   **`plan_feature.md`**: Analyze requirements and select personas.
-   **`spec_feature.md`**: Create detailed technical specifications.
-   **`implement_feature.md`**: Execute the coding loop.
-   **`verify_feature.md`**: Run tests and persona-based audits.

### Bugfix Lifecycle (Split)
-   **`identify_bug.md`**: Reproduce and isolate issues.
-   **`plan_fix.md`**: Design a regression-free fix.
-   **`implement_fix.md`**: Apply the fix.
-   **`verify_fix.md`**: Verify resolution.

### Maintenance
-   **`review_codebase.md`**: Analyze legacy code.
-   **`retrospect.md`**: Improve processes.
